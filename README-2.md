# CodeAlpha Python Programming Internship — Task 4: Basic Chatbot

A simple rule-based chatbot that responds to keyword-based user input.

## Features
- Recognizes greetings, farewells, and a few common questions
- Keyword matching (not exact-match only), so casual phrasing still works
- Graceful fallback reply for unrecognized input

## Concepts Used
`if-elif`, functions, loops, input/output

## How to Run
```
python chatbot.py
```

## Sample Conversation
```
ChatBot: Hi! I'm a simple rule-based chatbot. Type 'bye' to exit.

You: hello
ChatBot: Hi! How can I help you today?

You: bye
ChatBot: Goodbye! Have a great day.
```
